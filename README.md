# Synacor Challenge From OSCON 2012 By Eric Wastl

See more here and get a copy of the challenge binary :- https://github.com/Aneurysm9/vm_challenge


To run,

```bash
go run ./main.go ./challenge.bin

# or with a log level

LOG_LEVEL=info go run ./main.go challenge.bin

# LOG_LEVEL can be debug, info, warn or error
```
